# 🎓 Grade Giver

A modern and interactive **Grade Giver** web application that allows users to enter their percentage and instantly view their grade, performance, and motivational feedback. Built using **HTML, CSS, and JavaScript**, this project demonstrates the use of **if...else** statements while featuring a premium glassmorphism design inspired by the **Coral Wave** color palette.

---

## 📸 Preview

Enter your percentage, click **Check Grade**, and instantly receive:

- 📊 Animated Progress Circle
- 🎓 Grade
- 📋 Report Card
- 💬 Motivational Quote
- 🎉 Confetti Celebration (for A+)

## 🚀 How to Use

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Enter your percentage (0–100).
4. Click **Check Grade** or press **Enter**.
5. View your grade, report card, progress ring, and motivational message.
